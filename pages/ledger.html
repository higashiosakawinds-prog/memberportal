出納帳

Google Sheets 連携　スプレッドシートと同期するには、Apps Script WebアプリのURLを設定してください。
設定方法：スプレッドシートの「拡張機能 → Apps Script」で下記のスクリプトを貼り付け、ウェブアプリとしてデプロイし（アクセス：全員）、発行されたURLを入力してください。 URLは https://script.google.com/macros/s/.../exec の形式です。

Apps Script のセットアップ

以下のコードを Google Apps Script に貼り付け、「ウェブアプリとしてデプロイ」→「アクセスできるユーザー：全員」で公開してください。

// Google Apps Script — 出納帳連携スクリプト
// シート名は「出納帳」に設定してください。列順: 日付, カテゴリ, 摘要, 収支, 金額

const SHEET_NAME = '出納帳';

function doGet(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME);
  const rows  = sheet.getDataRange().getValues();
  const data  = [];
  for (let i = 1; i < rows.length; i++) {
    const row = rows[i];
    if (!row[0]) continue;
    data.push({
      id:          String(row[5] || i),
      date:        Utilities.formatDate(new Date(row[0]), 'Asia/Tokyo', 'yyyy-MM-dd'),
      category:    String(row[1]),
      description: String(row[2]),
      type:        String(row[3]),   // 'income' or 'expense'
      amount:      Number(row[4]),
    });
  }
  return ContentService
    .createTextOutput(JSON.stringify({ ok:true, data }))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME);
  const body  = JSON.parse(e.postData.contents);
  const id    = String(Date.now());
  sheet.appendRow([
    body.date, body.category, body.description,
    body.type, body.amount, id
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({ ok:true, id }))
    .setMimeType(ContentService.MimeType.JSON);
}
